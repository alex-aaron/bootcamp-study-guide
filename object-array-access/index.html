Object & Array Access

Arrays

It's important to understand the difference between arrays and objects. Fundamentally, arrays are numbered lists of like items. All arrays are zero-indexed, meaning each item is numbered starting at 0. So, for any array you create, you want to imagine there is an invisible index number next to each item. 

      
        const friends = ['Bethany Joseph', 'Ben Mason', 'Sean Thomas'];
        // indexes              0              1             2

If you want to access an item on an array, you have to access its index either literally or via reference using bracket notation. 

      
        const friends = ['Bethany Joseph', 'Ben Mason', 'Sean Thomas'];
        
        // accessing items using literal index values
        console.log(friends[0]); // LOGS => Bethany Jones
        console.log(friends[1]); // LOGS => Ben Mason
        console.log(friends[2]); // LOGS => Sean Thomas
      
    
      
        const friends = ['Bethany Joseph', 'Ben Mason', 'Sean Thomas'];
        
        // accessing items using references
        const targetIndex = 1;

        console.log(friends[targetIndex]); // LOGS => Ben Mason
        console.log(friends[friends.length - 1]); // LOGS => Sean Thomas

It's important to note that the principle of how you access values in an array is the same, even if the data gets more complex. Take an array of subarrays for example... 

      
        const nflDivision = [
          [
            "New Orleans Saints",
            "Tampa Bay Buccaneers",
            "Carolina Panthers",
            "Atlanta Falcons"
          ],
          [
            "Dallas Cowboys",
            "Philadelphia Eagles",
            "Washington Commanders",
            "New York Giants"
          ]
        ];

        console.log(nflDivisions[0][0]); // LOGS => 'New Orleans Saints'
        console.log(nflDivision[1][nflDivisons[1].length - 1]); // LOGS => 'New York Giants'

In the above example, nflDivisions[0] accesses a nested array of NFL team names. Since we know nflDivisions[0] represents that data, then we know that accessing the zero index of THAT will give us 'New Orleans Saints.
